## トップページ仕様

- ダーク/ライトモード切り替え（ボタン1つでトグル）
- `prefers-color-scheme` で初期テーマを検出、`localStorage` に保存
- CSS 変数 + `[data-theme]` 属性で切り替え
- lo-fi レトロ Apple 風デザイン
- FOUC 防止のためインラインスクリプトで CSS 読み込み前にテーマ適用
